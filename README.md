# 🎮 Gaming Arena

A responsive **Gaming Tournament Website** built using **HTML, CSS, and JavaScript**.
It showcases featured games, tournament schedules, and a contact section for users.

---

## 🚀 Features

* 🎯 Modern gaming UI design
* 📱 Fully responsive (mobile-friendly)
* 🎮 Featured games section
* 📝 Registration buttons for each game
* 📅 Tournament schedule table
* 📩 Contact form interface
* 🍔 Mobile sidebar navigation
* ✨ Smooth scrolling animations

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Grid, Media Queries)
* JavaScript
* Google Fonts
